1. id 선택자 (#) : 우선순위가 제일 높다! - 한 문서에 딱 하나만 존재할 수 있는 선택자 - 한 태그에 하나까지만 넣을 수 있다. 2. class 선택자 (.) - 한 문서에 여러개 존재할 수 있는 선택자 - 한 태그가 여러 클래스를 동시에 가질 수 있다. - 여러 태그가 하나의 클래스를 같이 가질 수 있다. 3. tag 선택자 - html 태그를 활용하는 선택자 - reset 할 때 주로 사용한다. - 디자인을 넣기위해 사용한다기 보다는 빼기위해 사용할 때가 더 많다. 위 선택자들을 조합해서 사용 가능하다. p.영업팀#박영희 { css속성 }
