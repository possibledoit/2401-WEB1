(3n+1) 이런 수열을 생각해보면 n = 0 부터 대입해보면 된다. n = 0 -> 1 n = 1 -> 4 n = 2 -> 7 p:nth-child(3n+1) 이라는 선택자는 1, 4, 7, 10, ... 번째에 등장하는 p 태그에 속성을 적용하겠다는 뜻이다. nth-last는 뒤에서부터 숫자를 세는 수열이다.(nth-child의 반전이다.) 뒤 : child, of-type 반응 선택자 :active -> 마우스 클릭하고 있을 때 :hover -> 마우스가 컨텐츠 위에 있을 때 자세한 내용은 day13의 수업 내용들 참고하기! 상태 선택자 컨텐츠의 상태에 따라서 속성을 지정해주는 선택자 :checked 체크박스에 체크가 되어있는가? :focus 컨텐츠가 포커싱되어있는가?(선택되어있는가?) tap키를 눌러보면 다음컨텐츠로 이동되는데, 테두리 있는 그 상태가 포커스 상태 물론 css로 테두리를 지울 수 있기 때문에 항상 테두리가 있는 것은 아닙니다. :enabled 컨텐츠가 활성화 되었을 때 :disabled 컨텐츠가 비활성화 되었을 때 day14 - state.html 참고하기!
